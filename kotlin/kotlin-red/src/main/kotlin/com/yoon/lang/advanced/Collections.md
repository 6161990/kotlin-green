# 01 Collections

* 코틀린 표준 라이브러리는 기본 컬렉션 타입인 `List` `Set` `Map` 을 제공한다.
* 컬렉션은 두 가지 종류로 나뉜다. 
  * 불변 컬렉션(immutable) : 읽기 전용 컬렉션
  * 가변 컬렉션(mutable) : 삽입, 수정, 삭제와 같은 쓰기 작업이 가능한 컬렉션
