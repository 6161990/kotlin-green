# SingleTon
* 클래스의 인스턴스를 하나의 단일 인스턴스로 제한하는 디자인 패턴
* 제약사항
  * 생성자를 private 로 숨긴다
  * getInstance() 라는 클래스의 단일 인스턴스를 반환하는 static 메서드를 제공한다
  * 멀티-스레드 환경에서도 안전하게 유일한 인스턴스를 반환해야한다 
* 구현방식
  * DCL Double Check Locking
  * Enum 싱글톤
